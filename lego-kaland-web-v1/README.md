# LEGO Kaland Web v1

Első működő weboldal-váz GitHub Pages-hez.

## Mit tud?

- Intro videó teljes képernyőn
- Videó vége után térkép
- 5 placeholder helyszín
- Popup ablakok a helyszínekhez
- `noindex,nofollow`, hogy keresők ne indexeljék

## Hova kerül az intro videó?

Tedd ide:

```text
assets/video/intro.mp4
```

## GitHub Pages

1. Töltsd fel a fájlokat a repository-ba.
2. Settings → Pages
3. Source: Deploy from branch
4. Branch: main
5. Folder: /root
6. Save
